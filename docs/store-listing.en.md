## App name

Chess Opening Trainer

## Short description

Train basic chess openings and continue the game with Stockfish.

## Full description

Chess Opening Trainer helps beginners practice basic chess openings in a fast and mobile-friendly way.

The app lets you:

- train opening lines move by move,
- choose your side,
- continue playing after the opening line ends,
- use a local Stockfish engine for hints and follow-up moves,
- practice without relying on an external backend server.

Key features:

- local opening database,
- opening training mode,
- free play mode,
- best-move hints,
- continue-playing mode with Stockfish,
- responsive mobile-focused interface.

The app is designed for comfortable phone-based training while keeping the same core project architecture used on desktop.

The app uses the local Stockfish chess engine running on the user's device. Position analysis and engine moves are processed locally.
Chess engine: Stockfish
